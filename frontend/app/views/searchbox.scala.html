Advanced Search Options

Advanced Search Options

Lucene's Query Syntax may be used to compose advanced searches and narrow searches to specific fields.

For example, to search for collections containing finders:

isCollection:true AND hasFinder:true
Or, to find resources supporting partial updates: 
method:partial_update

The following fields are available:

Field Description
keywordsthis is the default search field ("activities" and "keywords: activities" are equivalent) and includes many of the terms from the other fields to make simple search more convenient.
resourcethe name of a resource. E.g. "resource:countries"
clusterthe name of a cluster. E.g. "cluster:UscpProxyApi"
protocolmay be either "REST" or "CONTENT_SERVICE"
isCollectionmatches collection resources. E.g. "isCollection:true"
hasComplexKeymatches collections with complex keys. E.g. "hasComplexKey:true"
isAssociationmatches association resources. E.g. "isAssociation:true"
isSimplematches simple resources. E.g. "isSimple:true"
isActionSetmatches action set resources. E.g. "isActionSet:true"
isSubresourcematches subresources. E.g. "isSubresource:true"
hasFindermatches resources containing finders. E.g. "hasFinder:true"
finderthe names of finders on a resource. E.g. "finder:criteria"
hasActionmatches resources containing any actions. E.g. "hasAction:true"
hasEntityActionmatches resources containing actions that are at the 'entity' level. E.g. "hasEntityAction:true"
actionthe names of actions on a resource. E.g. "action:clear"
methodthe names of supported methods on a resource. Available methods are E.g. "method:partial_update"
assocKeysthe names of association keys on an association resource. E.g. "assocKeys:src"
isDeprecatedmatches resources that are marked as deprecated. E.g. "isDeprecated:true"
isInternalToSuperblockmatches resources that are marked as internal to their superblock. E.g. "isInternalToSuperblock:true"
hasDeprecatedmatches resources with methods/finders/actions that are marked as deprecated. E.g. "hasDeprecated:true"
hasTestMethodmatches resources with methods/finders/actions that are intended for use by tests only. E.g. "hasTestMethod:true"
isColoVariantmatches resources that are a d2 colo variant of another resource. Defaults to "isColoVariant:false" unless isColoVariant or coloVariant are provided.
coloVariantmatches that have a specific d2 colo variant suffix. E.g. "coloVariant:Master"
createdDateThe date the resource was first added to D2. E.g. "createdDate[20131200 TO 20131231]"
modifiedDateThe date the resource was last modified in to D2. E.g. "modifiedDate[20131200 TO 20131231]"
