Create permlink from a snapshot.json file

Upload a snapshot.json here to create a fully rendered permlink for it.



Example permlink: permlink:30344.

Snapshot files are located in the source directories of api modules after they are built (e.g. ligradle build or ligradle publishRestliSnapshot). For MPs, this is usually {api-project}/src/main/snapshot.

Your snapshot file will be stored in go/pastein and you will be redirected to a permlink URL that can be shared.
